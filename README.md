# Parcel Tool

## Overview

A logistics operations tool available in both PDA and Web versions for parcel processing, route assignment, and exception handling.

## Modules

* PDA Version: scan-based real-time processing
* Web Version: batch processing with route mapping and automation

## Core Features

* Route assignment by zipcode
* Wrong address handling (blind batch + 998 transfer)
* Driver reassignment
* Status update (202)
* Excel-based route configuration
